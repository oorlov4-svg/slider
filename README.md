📌 Project: Tab-Based Slider
📖 Description

This project is an interactive slider controlled by tab navigation, built using vanilla JavaScript, HTML, and CSS.

Instead of traditional arrows, users switch between slides by clicking on tabs. Each tab corresponds to a specific slide, making the interface intuitive and well-suited for structured content presentation.

The project demonstrates practical skills in DOM manipulation, state management, and UI synchronization.

🚀 Features
🧭 Navigation via tabs (click to switch slides)
🎯 Active tab highlighting
🔄 Smooth content switching
📱 Responsive layout
⚡ No external libraries (pure JavaScript)
🛠️ Tech Stack
HTML5
CSS3 (Flexbox / transitions)
JavaScript (ES6+)
📂 Project Structure
/project-root
│── index.html

⚙️ How It Works
Each tab is linked to a specific slide via index or data-attribute
On click:
Active tab class is updated
Corresponding slide is shown
Other slides are hidden
JavaScript ensures synchronization between tabs and content
